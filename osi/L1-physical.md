# L1 - 物理层 (Physical Layer)

## 职责
在物理介质上传输原始比特流，定义电气、机械、过程和功能特性。

## 核心功能

### 1. 比特编码与信号调制
- 将数据比特转换为物理信号（电、光、无线电波）
- 编码方式：曼彻斯特编码、4B/5B、8B/10B、PAM-4

### 2. 物理介质接口
- 定义连接器的形状、引脚布局
- 定义电气特性（电压、阻抗）
- 定义传输模式（单工/半双工/全双工）

### 3. 传输模式
| 模式 | 说明 | 示例 |
|------|------|------|
| 单工 | 单向传输 | 广播、电视 |
| 半双工 | 双向交替 | 对讲机、传统集线器 |
| 全双工 | 双向同时 | 电话、现代交换机 |

### 4. 拓扑结构
| 拓扑 | 说明 |
|------|------|
| 总线型 | 所有设备共享一条总线 |
| 星型 | 所有设备连接中心节点 |
| 环型 | 设备首尾相连形成环 |
| 网状 | 设备间多路径互联 |
| 树型 | 层次化星型组合 |

## 常见编码方式

### 曼彻斯特编码（10BASE-T 以太网）
```
数据:     1    0    1    1    0
信号:   ┌─┐  ┌──┐┌─┐ ┌─┐ ┌──┐
        │ │  │  ││ │ │ │ │  │
        └─┘  └──┘└─┘ └─┘ └──┘
        高→低=1  低→高=0
```

### 4B/5B 编码（100BASE-TX）
- 每 4 位数据映射为 5 位编码
- 保证足够的信号跳变用于时钟恢复
- 效率：80%

### PAM-4（25G/100G 以太网）
- 4 个电压级别，每符号 2 bit
- 更高频谱效率，但对噪声更敏感

## 常见物理层设备
| 设备 | 功能 |
|------|------|
| 中继器 (Repeater) | 放大/再生信号，扩展传输距离 |
| 集线器 (Hub) | 多端口中继器，所有端口共享冲突域 |
| 光纤收发器 | 电信号 ↔ 光信号转换 |
| 调制解调器 (Modem) | 数字 ↔ 模拟信号转换 |

## 物理层关键参数
- **带宽**：信道可传输的最高频率（Hz）
- **数据速率**：实际传输的比特率（bps）
- **衰减**：信号随距离减弱的程度
- **时延**：信号从发送到接收的时间
- **误码率 (BER)**：传输错误的比特比例
